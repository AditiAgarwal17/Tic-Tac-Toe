# tic-tac-toe
Tic-tac-toe (also known as noughts and crosses, or Xs and Os/"X'y O'sies"), is a paper-and-pencil game for two players, X and O, who take turns marking the spaces in a 3Ã—3 grid. The player who succeeds in placing three of their marks in a diagonal, horizontal, or vertical row is the winner.
For more details, visit- https://en.wikipedia.org/wiki/Tic-tac-toe

This is a multi-player game written using the Tkinter Module in PyCharm.
